## Post 1 — LinkedIn V3

**Opening line:** "Most AI automations that behave unpredictably aren't missing a better model. They're missing control flow."

**Recommended category:** Category C (Sketch Systems)

**Visual concept:**
A hand-drawn or sketch-style loop diagram showing the Think → Act → Verify cycle with the stop condition marked explicitly. The stop condition should be visually distinct — an exit point on the loop, not just a label. The diagram should look like something drawn while explaining the system to someone, not like something produced in Figma.

Optional: a simple annotation in a second color pointing to the stop condition exit with a short label ("missing in most implementations" or similar).

**Emotional tone:**
Operational clarity. Someone who knows this system well enough to sketch it quickly.

**Layout direction:**
Diagram centered or slightly left-weighted. Dark base. The diagram takes up ~60% of the frame. If any text, it's a single-line caption below the diagram (the post's opening claim or "Think → Act → Verify" only).

**Color direction:**
- Base: charcoal `#1C1C1E`
- Loop path and labels: off-white `#E8E8E4`
- Stop condition accent: muted amber `#C9862A`
- No other colors

**Explicitly avoid:**
- Polished flowchart with rounded boxes and drop shadows
- Any iconography (brain, robot, gears)
- Color gradients
- More than one accent color
- Text that duplicates the caption

**Recommended format:** Single image, 1080×1080 or 1200×627

**Does this post work without an image?**
Yes. LinkedIn text posts are viable. The image improves visual differentiation in feed, but the post is strong without it.

## Post 2 — Instagram V2

**Opening line:** "An AI that answers is a tool. An AI that loops is a system."

**Recommended category:** Category C (Sketch Systems) → Carousel

**Visual concept:**
5-slide carousel. Each slide is a single concept on a dark base.

```
Slide 1: "An AI that answers is a tool. An AI that loops is a system."
         [Primary text, large, centered. No diagram. Just the claim.]

Slide 2: "Think"
         [Phase label + one-liner: "plan before acting"]
         [Small loop diagram visible as a background element — position at Think]

Slide 3: "Act"
         [Phase label + one-liner: "one action only, then stop"]
         [Loop diagram — position at Act]

Slide 4: "Verify"
         [Phase label + one-liner: "did it work? If not, return to Think"]
         [Loop diagram — position at Verify, arrow looping back visible]

Slide 5: "The model isn't the problem. The control flow is."
         [Closing claim. Plain. Dark base. No diagram.]
```

The loop diagram on slides 2–4 should be the same sketch, with a highlight or accent indicating which phase that slide is on. This creates visual continuity across the carousel.

**Emotional tone:**
Calm, structured. The kind of thing that would appear on a whiteboard during a technical briefing — not a branding shoot.

**Layout direction:**
Each slide: 80% of frame is the primary text/label. The background loop sketch (slides 2–4) is muted, not dominant. Generous padding. No crowding.

**Color direction:**
- Base: charcoal `#1C1C1E`
- Primary text: off-white `#E8E8E4`
- Phase highlight (active phase per slide): muted amber `#C9862A`
- Inactive loop diagram: low-opacity off-white (~30%)
- No other colors

**Explicitly avoid:**
- Icons on each slide
- Progress bar or slide counter (visual clutter)
- Decorative separators between text and background
- Gradient backgrounds
- Emoji in visuals
- Closing slide with "follow for more" or any CTA

**Recommended format:** Carousel, 5 slides, 1080×1080 each

**Does this post work without an image?**
Less well than LinkedIn. Instagram is visual-primary. A plain text-only post would work but would underperform relative to a carousel on this content type.

## Post 3 — Facebook V3

**Opening line:** "If your AI workflow works fine once but breaks down over multiple steps — the issue is usually not the model."

**Recommended category:** Category B (Operational Screens) or no image

**Visual concept (Option A — with image):**
A real screenshot of an n8n workflow, a terminal log, or any automation tool output that shows a multi-step process. Lightly annotated — one arrow or circle in the accent color pointing to a specific node or step. The annotation implies "this is where it breaks" without labeling it explicitly. The screenshot should be real, not staged.

**Visual concept (Option B — typography):**
The quotable line from the post, set in clean type on a dark base:

> "I set up an AI automation" and "I have a reliable AI automation"
> are not the same statement.

Two-line format. First line slightly smaller than second. No decorative elements.

**Emotional tone:**
Option A: practical, grounded — looks like something from actual work
Option B: direct, slightly wry — the gap between these two things is the whole post

**Layout direction:**
Option A: screenshot fills frame, annotation is minimal and off to the side
Option B: text centered, generous padding, nothing else in frame

**Color direction:**
- Option A: whatever the screenshot's native color scheme is (dark terminal preferred); accent arrow in muted amber
- Option B: charcoal base, off-white primary text, no accent needed

**Explicitly avoid:**
- Generic workflow diagram illustration (use real screenshot or none)
- Anything that looks like a motivational quote card
- Stock imagery of people using computers
- Diagram that requires reading to understand (Facebook audience)

**Recommended format:** Single image, 1200×630

**Does this post work without an image?**
Yes, easily. Facebook text posts are viable and this post's scenario opener is engaging on its own. Option B (typography) only if the quotable line is strong enough to carry a visual — it is. Option A only if a real screenshot is available. No image is better than a staged or stock image.

## Visual Variety Check — Sprint 1

| Post | Category | Format | Accent |
|---|---|---|---|
| LinkedIn V3 | C — Sketch | Single image | Amber |
| Instagram V2 | C — Sketch | Carousel | Amber |
| Facebook V3 | B — Operational or none | Single image or none | Amber / none |

**Warning:** All three Sprint 1 posts lean toward Category C and amber accent. If all three are produced as specified, the visual set will look consistent but repetitive. Before producing these assets, consider:
- LinkedIn: could use Category A instead (the opening claim in clean type is strong)
- Facebook: strongly consider no image or Option B (typography)

This distributes visual weight across the sprint without manufacturing artificial variety.
