# Instagram Visual Assets

## What works

- **Carousel (Category C or A):** 5–7 slides, one idea per slide. The primary format for educational/operational content.
- **Single image (Category A):** Strong single claim on a clean dark base. Works when the opening line is the whole point.
- **Diagram (Category C):** Loop diagram, process map, sketch system. Works when the concept has a clear visual structure.

## Aspect ratios

- `1080 × 1080` (1:1) — standard, no cropping on any device
- `1080 × 1350` (4:5) — portrait, takes more screen space in feed, slightly higher attention
- Avoid 16:9 for feed posts — appears small and gets cropped

## Carousel structure (if used)

```
Slide 1: Hook / opening claim (strong enough to stop scroll)
Slide 2–5: Core content, one point per slide
Slide 6: Summary claim (NOT a restatement — new specific insight)
Optional Slide 7: Visual-only (diagram, map, sketch)
```

No "follow for more" slides. No "save this post" prompts.

## Text on image rules

- Primary text: legible at 150px thumbnail size
- Maximum 2 lines of text per slide (carousel) or 1–2 lines (single image)
- Use generous padding — text should never touch the edge
- Monospace or geometric sans only — no display or script fonts

## Pairing logic

Instagram caption is short. The visual carries primary weight.

- Post with three-phase breakdown → carousel with one slide per phase
- Post with a single sharp contrast → single image with that contrast in type
- Post with a process observation → diagram or sketch of the process

The visual leads. The caption provides operational context.

## What to avoid

- Screenshots that are unreadable at mobile size
- More than 3 lines of text on a single-image post
- Mixing visual categories within a carousel (all slides should feel like one set)
- Hashtags embedded in the visual
