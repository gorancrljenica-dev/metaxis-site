# facebook/approved/

Human-approved Facebook visuals. Passed feed-context test. Ready for manual publishing.

## What belongs here

- Final visual candidate that passed human review
- Feed-tested (checked at mobile render size in Facebook feed)
- Paired with an approved caption

## What does NOT belong here

- Raw or untested variants (those stay in raw/)
- Visuals that have already been deployed (those move to deployed/)
- LinkedIn or Instagram visuals

## When files move out

Move to `deployed/` immediately after the post goes live.
If a visual is approved but never used, it stays here with a note explaining why.

## Notes to accompany the asset

```
2026-05-21_facebook_agent-loopovi_APPROVED.png

Approved: YYYY-MM-DD
Caption: blog-001_facebook_APPROVED.md (V3)
Category: [A/B/C/D from VISUAL_RULES.md]
Base: [color], accent: [color]
Feed-tested: yes — mobile feed size
Notes: [any corrections from raw version]
```

## Facebook-specific

Facebook text-only posts are valid alternatives to visual posts.
If no visual is attached, this folder has nothing to hold — that is acceptable.
When a visual is used, verify it renders correctly in both mobile feed and desktop feed, as Facebook's rendering differs across devices more than LinkedIn.
