# GOOD Examples

Approved content that passed reviewer and evaluator checks.

## What belongs here

- Drafts approved for publishing (or approved and published)
- Content that received zero REVIEWER_RULES rejections
- Writing that is recognizably Metaxis: direct, calm, specific, non-generic
- Platform-native writing — not copy-pasted across channels

## Quality markers to look for

- Opens with a concrete observation, not a hook
- No rejected patterns (see REJECTED_PATTERNS.md)
- Specific operational claim, not abstract advice
- Platform format is correct (length, hashtag count, link usage)
- Passes the calibration test in TONE_GUIDE.md

## File naming convention

```
YYYY-MM-DD_platform_slug.md
```

Example:
```
2026-05-21_linkedin_agent-loopovi.md
2026-05-21_instagram_agent-loopovi.md
```

## Sprint discipline

Every sprint must produce at least one GOOD example.
If no draft passes review, the sprint is not done.
